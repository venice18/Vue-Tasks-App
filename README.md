# TaskApp

A clean and lightweight task management application built with Vue.js. TaskApp helps users organize their daily activities by creating, tracking, updating, and completing tasks through a simple and intuitive interface.

---

## Overview

TaskApp is designed to improve personal productivity by keeping task management simple and distraction-free. It focuses on speed, usability, and a clean user experience.

Whether you're managing daily to-dos or planning bigger goals, TaskApp gives you a straightforward way to stay organized.

---

## Features

* Add new tasks instantly
* Mark tasks as completed
* Edit existing tasks
* Delete tasks you no longer need
* Clean and responsive UI
* Fast and lightweight performance

---

## Built With

* Vue.js (Frontend framework)
* Vite (Development build tool)
* JavaScript (Core logic)
* HTML & CSS (UI structure and styling)

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/taskapp.git
```

Navigate into the project:

```bash
cd taskapp
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open in your browser:

```
http://localhost:5173
```
